Phonotactics & Stress

Template (C)(C)V(C)(C), with the following common patterns:

  • Frequent: CVC, CVCVC, CVCVCVC
  • Onsets: single consonants; note that lh- (É¬) and tl- (Æ›) are single consonants (written with two letters). True clusters include mr-, ml-, dr-, kl-, gl-...

Examples: lor (CVC), thim (CVC), loelor (CVCVC), tlan (CVC with single tl)

Stress

  • Primary: penultimate syllable (or the only syllable). This matches all stressed IPA entries in the dictionary.
  • Secondary: may appear earlier in longer words; IPA in entries marks primary stress with Ëˆ and often shows syllable breaks with ..
